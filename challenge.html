Ideation

As always i use The value proposition canvas because this tool make me easy to determine what a solution from the problem or how to catch the hope

Let me describe my ide.

  • First problem is He is not fluent in English nor bahasa, so that i try Use language that is easy to understand and simple
  • Second problem is sometimes when it is expected with several choices, people will get confused which one to choose. Are u remember when i tell u abaout my favorite UI is gojek? Because gojek is give me choose when i want to buy a food, so i not confuse again. So the solution is there are best product features so that customers can consider their choices.
  • Then the problem is In this case, Yakato already likes the design of the shop. To be honesly if i want, i can just make Yakato only can order wallet with design from suplier. But when i think again, what if Yakato has his own design? So that i make one fiture more. There is a feature to use your own design
  • Last problem is sometimes a customer wants to customize the model and shape of the wallet, so that there are features to request shapes and models according to customer needs
  • And dont forget Yakato hope. Yakato want buy custom wallet items with beautiful balinese design easily, so i want to make clear CTA, so that Yakato can guess what will happen next
